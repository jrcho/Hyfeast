    EmbeddedLine

    EmbeddedLine 요소는 매립된 텐던 또는 철근을 모사할 수 있는 요소로서, EmbeddedLine 요소 타입, EmbeddedLine 단면 타입으로 정의할 수 있다. EmbeddedLine 단면을 정의할 때 요소의 프로파일을 스프라인 기하형상으로 정의한다. 요소의 개별적인 특성치로 호스트 요소집합을 정의해야 한다. 적용 가능한 호스트 요소는 다음과 같다.

    • 2차원 보 요소군(Beam2D) : B2D2H, B2D2MH
    • 3차원 보 요소군(Beam3D) : B3D2H, B3D2MH
    • 쉘 요소군(Shell) : S3F, S4F, S3, S4
    • 솔리드쉘 요소군(CShell) : CS6, CS8
    • 2차원 평면 솔리드 요소군(Plane) : CPE4, CPS4, CAX4, CPS3, CPE3, CAX3
    • 3차원 솔리드 요소군(Solid) : C3D8, C3D8I, C3D6

    EmbeddedLine 요소에 적용하는 호스트 요소집합은 동일한 요소군으로 구성되어야 한다. 텐던 해석시에는 *State 명령으로 텐던의 프리스트레스 하중을 부과할 수 있다.

    ▪ 주의 - EmbeddedLine 요소는 질량행렬을 계산하지 않는다(강성에만 기여) - 철근 모델링은 Truss 요소(T3D2)와 Embedded 구속조건으로도 가능하다. 이 경우 직선 형태이고, 프리스트레스 등은 부과할 수 없다. - EmbeddedLine 요소는 1개의 절점을 사용하지만, 스프라인 형상의 기준점으로만 사용한다. 그 절점에서 자유도를 정의하지 않는다.

    *Element, Type=EmbeddedLine

    Define EmbeddedLine element

    *Element, Type=EmbeddedLine, ELSet=elset
  id, n1, S=sec, L=spline, H=hostElset
    First dataline and subsequent lines
    • L=spline: Spline defining the profile
    • H=hostElset: Host elset
    Specifications
    • No. of nodes: 1
    • Fields: ED=[ED.X ED.Y ED.Z] and uniaxial material field at Gauss point
    • Compatible section: EmbeddedLine
    • Active DOFs: None

    ▪ Notes The n1 node is used only as a reference point for defining the geometry and does not contribute to the stiffness or mass matrix of the element.

    *Section, Type=EmbeddedLine

    Define the properties for EmbeddedLine element

    *Section, Type=EmbeddedLine, Name=name
 material, area, ngauss, maxDiscretizationLength
 mu, kappa
    Keyword line
    • Name=name: Section name. Section names must be unique.
    First dataline
    • material: material (required)
    • area: area (required)
    • ngauss: No. of integration points for spline subdivision (optional, default is 4).
    • maxDiscretizationLength: Maximum allowed discretization length between Gauss points (optional,
    Second dataline (optional)
    • mu, kappa: Curvature and wobble friction coefficients (optional, default 0., 0.).

    The spline (*Construction, TYPE=Spline) is subdivided into segments between sampling points. Each subdivision is integrated using Gauss–Lobatto quadrature with ngauss integration points. To ensure accurate integration, a sufficient number of points is required, and this is controlled by the variable maxDiscretizationLength. maxDiscretizationLength specifies the maximum spacing allowed between adjacent integration points, and the actual spacing is set smaller than this value. If the parameter is not specified (i.e., set to 0), it is automatically assigned as one quarter of the smallest edge length of the host element.
