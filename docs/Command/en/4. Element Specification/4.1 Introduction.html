    Overview

    The elements are created using the *Element command. The number of nodes connected to each element is determined by the element type. Element properties include Section properties, which are commonly used across various elements, and Distribution properties, which are applied directly to individual elements. In addition, special elements such as Cable elements and Tendon elements have field values that can change depending on the analysis stage. The table below summarizes the corresponding Section properties, Distribution properties, and field values for each element type.

    Cable Cable Beam (R) BeamEndRelease CableLength

    B3D2H Bernoulli beam Beam (R) BeamEndRelease
    B2D2MH 2D Timoshenko beam Beam (CS) BeamCS
    B3D2MH Timoshenko beam Beam (SF) MCKElementScaleFactor
    Spring 2-node spring/damper MCK (SF) MCKElementScaleFactor
    EarthSpring 1-node spring/damper MCK (SF) MCKElementScaleFactor
    PointMass 1-node mass MCK (SF) UCS
    MovingSpring Moving spring MovingSpring (P) MovingSpringInitialPosition Speed

    S3F 3-node flat shell Shell
