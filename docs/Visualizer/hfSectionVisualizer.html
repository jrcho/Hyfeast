    hfSectionVisualizer 상세 사용자 매뉴얼 (단면해석 시스템 편)

    본 문서는 보, 기둥 및 교각 단면 등 콘크리트 및 복합 단면의 **단면 성능 성능(M-Curvature, PM 곡선)**을 도출하는 내부 솔버 hfSectionAnalyzer의 입력을 책임지고, 해석 파괴 과정을 시뮬레이션 해주는 전용 그래픽 도면 인터페이스 **hfSectionVisualizer**의 완전판 실무 가이드입니다.

    1. 섹션 뷰어의 도면 인프라 구성

    hfSectionVisualizer는 AutoCAD와 유사한 2D 평면 제도 환경을 기본 베이스로 깔고 들어갑니다.

    1.1 메인 평면 캔버스 및 스마트 그리드 스냅뷰

    • 화면 중심 (0,0) 기하학적 도심 마커를 베이스로 설계 외곽 테두리를 그립니다.
    • 눈에 보이지 않는 점들을 도식해 나갈 때 발생할 소수점 오류를 막기 위해 그리드 스냅 (Snap to Grid) 토글 기능을 켭니다. 1cm, 5cm 단위 등 Settings 다이얼로그에서 설정한 정수 베이스 간격으로 마우스 포인터가 격자 교차점을 콱콱 낚아채 정밀 기하 도면의 무결성을 100% 방어합니다.

    1.2 모델 아카이브 트리 (Model Tree View)

    생성된 모든 단면 덩어리와 재료들의 족보가 화면 좌측에 트립 폴더형으로 꽂힙니다. - Regions (폐합된 면적 윤곽선들): 사각형, H빔 형상 면적 파츠들. - Holes (타공 영역): 영역에 뚫어놓은 공허(빈공간). - Materials / Rebars: 재료 물성치 모델 목록과 철근 다발 목록. - 트리에서 오입력된 객체를 찾아 더블클릭하면, 우측의 "파라미터 커맨드" 탭이 열려 소수점 이하 수치를 키보드 텍스트로 보정 수정해낼 수 있습니다. (설계 변수화 기능의 정수).

    2. 작업 포맷 및 물리 단위의 지배 (Unit Dialog)

    단면 계수는 단위 오차 하나에 모멘트 결과값이 \(10^6\) 배 널뛰기하므로 가장 먼저 엄수해야 할 설정 탭입니다. - [Tools] -> Unit Settings: 모델링 제도 단위를 밀리미터(\(mm\)), 센티미터(\(cm\)) 중 무얼 쓸지 결정. - 해석 과정 시나리오에 쓰일 외력 하중 단위를 뉴턴(\(N\)), 혹은 킬로뉴턴(\(kN\))으로 박제합니다. 여기서 셋업된 단위를 기준으로 이후 뽑히는 모든 그래프 라벨의 \(MPa\) (응력) 축 텍스트가 자동 환산 맵핑됩니다.

    3. 기하학 단면 프로파일 제도와 재질 접착 (Pre-processing)

    3.1 콘크리트 및 베이스 외곽 다각형 성형 툴

    상단 Draw 위젯 툴 모음을 이용해 단면 콘크리트 덩이, 프레임 매스를 윤곽선으로 포획합니다. - Rect / Circle (기본 도형): 마우스 왼쪽 클릭 + 시프트 드래그로 반경을 직관적으로 뽑아내는 외부 스킨 툴. - Draw Polygon (다각형 작도): 모따기 된 우물통 교각 등 자유형 폐합(Closed) 경계를 만듭니다. 순서를 따라 틱틱 모서리를 스냅 찍고 종착점에서 더블클릭을 때리면, 내부에 옅은 회색이 채워지며 "물리적 단면 Region 객체"가 선언 완료됩니다. - Add Hole (빈 공간 도어 파기): 파이프, 박스 거더 같이 배를 갈라 속을 비워야 하는 단면 구획 용도. 먼저 생성되어 있던 외부 Region을 선택한 뒤 이 Hole 버튼으로 안에 윤곽을 치면, সেই 면적 만큼 교집합 배제(Boolean Minus) 처리되는 영리한 도구입니다.

    3.2 물성(Material) 선언 및 배정 맵핑

    1. Material Widget 발동: 트리 뷰 우클릭 -> Add Material. 탄소성 선형 빔 강재치, 압축 300MPa 파라볼라 콘크리트 모형 커브 함수 등을 체크/입력 선언 후 텍스트 이름짓기.
    2. 캔버스에 그린 Region 덩이를 클릭 -> 우측 프로퍼티 다이얼로그의 Material 콤보박스 리스트를 떨궈 방금 그 이름들로 심어 매핑을 완료합니다.

    3.3 철근 메시 그룹 배치 자동화 (Rebar Array Placement)

    압축만 버티는 콘크리트 대신 인장면을 지지하는 보강 철근(Rebar) 군집 배치기 툴 세트입니다. (각 철근은 고유 텍스트 입력창에서 **자신의 면적 \(A_s\) 와 재료 응력 속성**을 기안받아 승인되어져야 솔버에 전달됩니다). 1. Point Rebar: 단독 철근 X, Y 좌표 알박기. 2. Line Array Pattern: 빔 하부 인장 라인에 선 긋듯이 마커를 당겨, 시작과 끝 노드 사이에 "철근 N 가닥을 자동 등분 배열하라"는 리니어 복사 배치 명령. 3. Circular Array Pattern: 후프 파이프 기둥용 띠철근. 원호 곡률 반경(R) 및 등간격 각도를 입력하여 환상형 빙 둘러치기 자동 오토 마킹의 핵심 지원 도구. - 시인성 팁: 철근이 십자가 형태면 잘 안 보입니다. Settings -> Rebar Representation 옵션을 켜서 면적이 비례 반영된 '꽉 들어찬 빨간 타원(Filled Circle)' 모양으로 렌더링을 전환해 오류를 육안 필터링하십시오.

    4. 메쉬 포텐셜 분할 및 단면 상수 표출 (Solver Prep)

    윤곽 스케치가 다차원 연산으로 역학 변환을 일으키는 엔진부입니다. - 메쉬 번개 마크 (Mesh Gen): 메뉴 바 메쉬 클릭. 윤곽 단면 내부에 촘촘한 그물 격자망(Planar Face) 평면 요소 셀들을 삼각/사각 패턴으로 작렬 파티셔닝 합니다. 메쉬가 잘게 썰릴 수록 곡률 비선형 해석의 오차율은 줄고 계산 타임은 늘어납니다 (Max Size 파라미터 조절 권장). - 단면 상수 팝업 추출: 메쉬 성공과 동시 하단 텍스트 콘솔(혹은 팝업) 트레이에 공학 기본 스펙 족보가 실시간 계산 리포팅 됩니다. (단면 구적도 면적 \(A\), 질량 중심 이동 도심 벡터, 2차 관성 단면 모멘트 \(Iy/Iz\), 횡/비틈 전단 댐핑 상수)

    5. 해석 케이스 가동과 동적 시뮬레이터 차트 (Run & Post)

    5.1 해석 통제 모드 스텝 (Step Maker)

    특정 비선형 파괴를 유도할 하중 제어 방식을 디자인하여 트리에 탑재합니다. - NK 모드 : \(P\)(축력) 버튼을 잠금으로 누르고 일정 값을 먹인 채 강제로 단면 극한 허용 곡률 각도를 와이어로 비틀어 꺾어 나갈 때 쓰는 한계 모멘트 산출 모드. - NM 모드 : 편심 모멘트로 단면의 층고 부러짐을 계산. - 각 증분 스텝(Increment)이 너무 폭주하지 않도록 \(0.1e-5\) 씩 점차 증분 하중을 부여토록 꼼꼼히 텍스트 입력합니다.

    5.2 파괴 애니메이터 등고선 맵 (Result View)

    해석 EXE 바이너리가 구동 후 결과를 뱉어내면 도면 뷰 모드가 응력 변형률 등고선 지도(Contour 맵) 모드로 위상 시프트됩니다. - 하단 타임 슬라이드 바를 끌면 하중 증가 시 단면 인장측부터 균열이 나 퍼런색 스펙트럼에서 \(\rightarrow \rightarrow \rightarrow\) 시뻘건 최대파괴점(Max stress limit)으로 번지며 철근이 항복해 붕괴되는 현상을 비디오로 적나라하게 점층 모니터링 할 수 있습니다.

    5.3 이력 성능 평가 플롯 차트 위젯 (hfChartControlWidget의 위력)

    모든 설계서 컴플라이언스를 인증하기 위해, 이 시뮬레이션 데이터를 X-Y 라인 공학 꺾은선 차트로 바꿔주는 최고급 위젯 Show Chart 모드를 구동합니다. 1. 모멘트-곡률 선도 차트 (M-Curvature Graph): 단면이 비틀려 파괴되기 전 피크 모멘트 허용 한계 산과 이후 폭락하는 에너지 소산 그래프의 선도를 도식화합니다. 2. PM 붕괴 상관도 영역 차트 (Interaction PM Envelope): 기둥이 받을 수 있는 수십 ~ 수백 개의 압축력, 인장력 하중 케이스 연속을 다다다닥 백그라운드 해석 돌려서 파괴 한계점 좌표들을 모조리 수집. 이를 이어 2D 다각형 루프인 포락선(안전 바리게이트 존) 영역을 도면에 도식시켜 단면 편심에 대한 컴플라이언스 기준 가이드를 한방에 보여줍니다. 3. Data 엑스포팅 테이블: 브라우저 차트에 그은 수백 개의 XY 좌표를 Copy Table Data 팝업을 통해 클립보드로 전송, 엑셀 창에 붙여넣기(Paste) 연동을 다이렉트로 지원하여 문서 작업 효율을 무한대 증폭시킵니다.
